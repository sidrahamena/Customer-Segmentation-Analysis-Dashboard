# Customer Segmentation Dashboard

An interactive Power BI dashboard designed to break down retail customer profiles, map out geographic clusters, and discover the hidden factors that drive high-value purchasing behavior.

## 📊 Quick Peek
The dashboard focuses on user behavior and demographic trends across the dataset:
* **Overview KPIs:** Tracking unique footprint metrics—793 total customers across 529 distinct cities, generating 4,922 unique orders across 1,861 products.
* **Segment & Behavior Analysis:** Features a breakdown of customers by product category, a leaderboard of top-spending individual customer IDs, and an order timeline chart.
* **Advanced Features:** Includes a Power BI map visual tracking regional performance alongside an AI-powered *Key Influencers* module identifying major purchase drivers.

## 🛠️ Built With
* **BI & Visualization:** Power BI
* **Data Source:** Superstore Sales dataset from Kaggle
